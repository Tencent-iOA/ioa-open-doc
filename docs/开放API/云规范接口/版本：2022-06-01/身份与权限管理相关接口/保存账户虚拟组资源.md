## 2. 输入参数


| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| ResourceList.N | 是 | Array of [ResourceDataReq](/开放API/云规范接口/版本：2022-06-01/数据结构.md#ResourceDataReq) | 资源集 |
| VirtualGroupId | 是 | Integer | 账户虚拟组Id(只支持32位) |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| RequestId | String | 唯一请求 ID，每次请求都会返回。定位问题时需要提供该次请求的 RequestId。|
