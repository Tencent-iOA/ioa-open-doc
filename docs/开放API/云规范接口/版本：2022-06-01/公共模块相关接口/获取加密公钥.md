## 1. 接口描述

## 3. 输出参数

|---------|---------|---------|
| Data | [DescribePublicEncryptKeyData](/开放API/云规范接口/版本：2022-06-01/数据结构.md#DescribePublicEncryptKeyData) | 获取公钥加密的key数据<br/>注意：此字段可能返回 null，表示取不到有效值。|
