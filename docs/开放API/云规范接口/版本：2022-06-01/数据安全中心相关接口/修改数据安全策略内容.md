## 1. 接口描述

修改数据安全策略内容,私有化调用path为：capi/DLPOpenApi/ModifyDLPPolicyContent 8.6P1版本开始支持

## 2. 输入参数

| 参数名称 | 必选  | 类型 | 描述                                  |
|---------|-----|---------|-------------------------------------|
| Name | 是   | String | 策略名称;Required:true<br/>示例值：扫描       |
| Description | 否   | String | 描述<br/>示例值：扫描办公设备                   |
| Status | 否   | Integer | 启用停用状态 0停用 1启用(只支持32位)<br/>示例值：1    |
| PolicyId | 是   | Integer | 策略Id(只支持32位)  <br/>示例值：1    |
| DataLevelList |  否  | Array of String | 数据级别列表,全部<br/>示例值：[S1,S2,S3,S4]   |
| RuleId | 否 | Array of Integer | 规则ID |
| ControlChannel | 否 | Array of String | 管控通道 |
| ControlChannelType | 否   | Array of Integer | 管控通道类型 |
| Type | 是 | String | 告警:DlpAlarmRule;管控:DLPControlScope;拦截:DLPIntercept;通道白名单:DLPWhiteChannel|
| Data | 是  | String  | 策略内容 | 
| RiskLevel | 否 | Integer | 风险分数 |
| RiskName  | 否 | String  | 风险名称 |
| BusinessId | 是 | Integer | 业务ID DlpAlarmRule:252;DLPControlScope:253;DLPIntercept:255;DLPWhiteChannel:257|


## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| Data | [ModifyDLPPolicyContentRspData](/开放API/云规范接口/版本：2022-06-01/数据结构.md#ModifyDLPPolicyContentRspData) | 业务响应数据<br/>注意：此字段可能返回 null，表示取不到有效值。|
| RequestId | String | 唯一请求 ID，由服务端生成，每次请求都会返回（若请求因其他原因未能抵达服务端，则该次请求不会获得 RequestId）。定位问题时需要提供该次请求的 RequestId。|
