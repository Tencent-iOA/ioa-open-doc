## 1. 接口描述

查询文件下载链接,私有化调用path为：capi/DLPOpenApi/DescribeDLPFileDownloadUrl 8.6P1版本开始支持

## 2. 输入参数

| 参数名称 | 必选  | 类型 | 描述                                  |
|---------|-----|---------|-------------------------------------|
| FileName | 是   | String | 文件名称      |
| FileMd5 | 否   | String | 文件md5                   |
| ModuleType | 是   | String | 业务类型, DLP业务时传ModuleType=DLP |
| CosId | 是   | Integer |  对象存储Id |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| Data | [DescribeDLPFileDownloadUrlData](/开放API/云规范接口/版本：2022-06-01/数据结构.md#DescribeDLPFileDownloadUrlData) | 业务响应数据<br/>注意：此字段可能返回 null，表示取不到有效值。|
| RequestId | String | 唯一请求 ID，由服务端生成，每次请求都会返回（若请求因其他原因未能抵达服务端，则该次请求不会获得 RequestId）。定位问题时需要提供该次请求的 RequestId。|
